# 什么是PolarDB db-monitor

db-monitor是一个轻量级灵活的数据采集Agent，可以采集主机和实例的性能数据。它是一个插件化采集进程，运行在物理机或虚拟机上，可以采集该机器上所有实例（容器化和非容器化实例）的秒级性能数据和日志数据。

db-monitor经受了阿里云线上真实环境的考验。
